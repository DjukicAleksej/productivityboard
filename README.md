Personal Dashboard

A sleek, interactive personal dashboard web app designed to boost productivity and track daily goals. Features include a dynamic greeting, ambient clock, quote of the day, todo list, pomodoro timer, and productivity tips—all wrapped in a modern glassmorphic UI.

Features

Dynamic Greeting: Animated typing effect with blinking cursor displaying “Hello, Human/Scholar/Cool Person/Brochacho”.

Theme Toggle: Switch between light and dark modes with a stylish button fixed on the page.

Ambient Clock: Always-visible clock with current time and date floating in the top-right corner.

Quote of the Day: Random quotes with a button to fetch the next one.

Daily Goal Tracker: Set and complete your ONE goal for the day.

Todo List: Add, edit, and delete tasks with visual completion indicators.

Productivity Stats: Tasks done, remaining tasks, and day streak counter.

Pomodoro Timer: 15-minute focus sessions with pause, restart, and custom time options.

Productivity Tips: Quick cards with useful productivity hacks.

Installation

Clone this repository:
``` bash
git clone https://github.com/DjukicAleksej/productivityboard.git
```

⚠️ Note: This project was originally in another corrupted Git repository. It was recovered and moved to this new repo for proper version control.

Navigate to the project folder:
``` bash
cd personal-dashboard
```

Open index.html in your browser or use a local server for full functionality.

Usage

Toggle theme using the circular button on the top-left.

Watch the dynamic greeting and ambient clock.

Set your daily goal, manage todos, and track your productivity stats.

Use the Pomodoro timer to focus in sprints.

Click "Next" to see a new motivational quote.
